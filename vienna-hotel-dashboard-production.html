Setup Required

To enable live price fetching from Google Hotels, you need to run the backend API server.

# Install dependencies:

pip install flask flask-cors requests beautifulsoup4

# Run the API server:

python google_hotels_scraper.py

The server will run on http://localhost:5000 and automatically fetch prices from Google Hotels.

How to Use

  1. 1. Make sure the backend API is running (see setup above)
  2. 2. Select check-in and check-out dates
  3. 3. Choose number of guests (1-4 people)
  4. 4. Click "Fetch Live Prices" to scrape Google Hotels
  5. 5. Enter your hotel's price to see comparison
  6. 6. System alerts you when ±20% from average
